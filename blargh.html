Terminate Dead Session (TDS)

A shell script for Sys Admins to keep on their remote system

Usage

Just log into the system and run the script to clean up your old processes

$ ./tds

Purpose

This script can be used/modified for when a system is clogged up with bad processes. If the user can still remote into the box they could use this script to automate cleaning up the system. Thus allowing it to return to a more stable state. Please feel free to use and modify this script for your needs!
